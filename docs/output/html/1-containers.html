Containers

What it is

Containers are just a plain set of divs, they can contain items.

Specs

CSS Class: .x-container

Ext.container.Container

Extends from Ext.Component

  • height

  • width

  • margin

  • padding

  • border

  • style

  • hidden (true / false)

Example

Ext.create('Ext.container.Container', {
    html: 'Hello World',
    width: 400,
    renderTo: Ext.getBody(),
});
