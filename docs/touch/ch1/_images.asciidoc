=== Images

==== What it is
An HTML image tag (optionally wrapped in div elements)
to display images.

==== Specs

===== CSS Class: +.x-img+

===== +Ext.Img+
Extends from +Ext.Component+

* +src+
* +alt+
* +autoEl+
* +height+
* +width+
* +margin+
* +padding+
* +baseCls+
* +imgCls+
