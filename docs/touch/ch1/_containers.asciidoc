=== Containers

==== What it is
Containers are just a plain set of divs, they can contain items.

==== Specs

===== CSS Class: +.x-container+

===== +Ext.container.Container+
Extends from +Ext.Component+

* +height+
* +width+
* +margin+
* +padding+
* +border+
* +style+
* +hidden+ (true / false)

http://docs.sencha.com/extjs/{vers}/#!/api/Ext.container.Container
